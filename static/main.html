Vial

Welcome to Vial! If you're seeing this page, that means it works! Here are some useful instructions for using Vial.

Page Content

Want to serve a basic page without any server-side logic? All you have to do is open your config.json and add an object that looks like this to the handlers key:

{"type": "static", "path": "/basic-page", "fileName": "cool-page.html", "contentType": "text/html"}

Then, write some HTML in cool-page.html, restart this server, and visit /basic-page. You can serve images, videos, text files, and anything you want by adding the file and changing the contentType to the mime type of the file.

Handlers

Making a handler is pretty easy. See the handler in handlers/base.js for reference. The request and response objects provided are from express, so anything you can do there should work. Once you have created the file for your handler, you need to register it in the config.json by adding {"type":"custom","handler":"base.js"} to the handlers key.
